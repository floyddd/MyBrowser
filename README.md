MyBrowser
===========
A simple iPhone Browser built on Xcode 5 for iOS 7 which uses ARC.

Features:

- iOS7 support
- Reload page
- URL validation
- Adding Bookmark
- Save history of URLs.
- Save bookmark and history data after app quits.
- Check internet connection and display alert if no connection.
